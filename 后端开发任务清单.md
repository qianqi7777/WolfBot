# AI狼人杀后端开发任务清单

以下任务按“先服务骨架、再房间流程、后实时通信、最后规则与 AI”的顺序组织。

## 一、工程初始化

- [ ] 初始化 FastAPI 工程
- [ ] 配置 Python 虚拟环境
- [ ] 安装基础依赖：fastapi、uvicorn、pydantic、sqlalchemy、alembic、redis、pytest
- [ ] 安装规范依赖：ruff、black
- [ ] 配置环境变量文件
- [ ] 建立基础目录结构

完成标准：服务可以启动，健康检查可访问。

## 二、基础规范

- [ ] 统一配置管理
- [ ] 统一日志输出
- [ ] 统一异常处理结构
- [ ] 统一响应格式
- [ ] 统一错误码定义
- [ ] 统一类型与命名风格

完成标准：项目具备一致的工程骨架。

## 三、模型与数据层

- [ ] 设计 Game、Room、Player、Action、Vote、Result 模型
- [ ] 建立数据库迁移配置
- [ ] 建立仓储层
- [ ] 建立 Redis 缓存封装
- [ ] 建立领域枚举与常量

完成标准：核心数据可以落库并被稳定读取。

## 四、房间接口

- [ ] 创建房间接口
- [ ] 加入房间接口
- [ ] 房间状态查询接口
- [ ] 开始游戏接口
- [ ] 房间成员管理接口

完成标准：前端可以完成创建与加入流程。

## 五、WebSocket 层

- [ ] 建立连接管理器
- [ ] 建立广播机制
- [ ] 建立消息格式模型
- [ ] 建立断开清理逻辑
- [ ] 建立在线玩家映射

完成标准：可以稳定向多个客户端推送事件。

## 六、对局流程
