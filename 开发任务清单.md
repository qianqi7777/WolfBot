# AI狼人杀（Vue3全生态）前端开发任务清单

以下任务按“先骨架、再通信、后流程、最后优化”的顺序组织。建议采用勾选制推进，完成一项勾一项，避免边写边散。

## 一、项目初始化

- [ ] 初始化 Vue3 + Vite + TypeScript 工程
- [ ] 选择并固定包管理器，推荐 pnpm
- [ ] 安装基础依赖：Pinia、Vue Router、Element Plus、Axios
- [ ] 安装开发依赖：ESLint、Prettier、typescript 相关插件
- [ ] 配置 `.env.development` 与 `.env.production`
- [ ] 配置路径别名，例如 `@` 指向 `src`
- [ ] 添加基础脚本：dev、build、preview、lint

完成标准：项目能够成功启动，且代码检查可以跑通。
