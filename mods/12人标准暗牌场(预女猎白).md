# 标准12人场（预女猎白）规则流程

12人标准场采用**预女猎白**板型，为**暗牌局**且**有警长**，采用**屠边**规则判定胜负。以下是完整规则流程：

---

### 一、基础配置

| 阵营 | 身份 | 数量 | 核心技能 |
|------|------|------|----------|
| **狼人阵营** | 普通狼人 | 4 | 每晚可袭击一名玩家，白天可自爆 |
| **好人阵营** | 预言家 | 1 | 每晚可查验一名玩家身份（好人/狼人） |
| | 女巫 | 1 | 拥有解药（救被刀玩家）和毒药（毒杀任意玩家），每晚最多用一瓶药，**不可自救** |
| | 猎人 | 1 | 被狼人杀害或投票放逐时可翻牌枪杀一名玩家，**被毒杀不可开枪** |
| | 白痴 | 1 | 被投票放逐时可翻牌免疫出局，之后可发言但不能投票，狼人仍需击杀一次 |
| | 平民 | 4 | 无特殊技能，通过发言和投票帮助好人阵营 |

---

### 二、胜利条件

- **狼人阵营**：达成**屠边**——杀死所有神职（预言家、女巫、猎人、白痴）或所有平民
- **好人阵营**：放逐所有狼人出局

---

### 三、游戏完整流程

#### 1. 游戏准备阶段
- 系统随机分配12名玩家的身份牌，玩家查看自己身份
- 系统宣布"游戏开始，天黑请闭眼"，所有玩家闭眼进入夜晚阶段

#### 2. 夜晚行动阶段（按固定顺序）
1.  **狼人请睁眼**（约60秒）：4名狼人互相确认身份，共同商议并选择一名玩家作为袭击目标（刀人），随后闭眼
2.  **预言家请睁眼**（约20秒）：预言家选择一名玩家进行查验，系统通过手势告知其身份（好人/狼人），随后闭眼
3.  **女巫请睁眼**（约20秒）：系统告知女巫当晚被刀玩家（若有），女巫可选择：
    - 使用解药救活被刀玩家
    - 使用毒药毒杀任意一名玩家
    - 不使用任何药剂（解药仅首夜可救自己？不，官方规则明确**女巫不可自救**）
    女巫每晚最多使用一瓶药，随后闭眼
4.  **猎人请睁眼**（约5秒）：系统告知猎人当前状态（能否开枪），随后闭眼
5.  **白痴请睁眼**（仅首夜，约5秒）：白痴睁眼让系统确认身份，随后闭眼
6.  系统宣布"天亮了，请所有玩家睁眼"，进入白天阶段

#### 3. 警长竞选阶段（第一天白天特有）
1.  **上警报名**（15秒）：玩家选择是否参与警长竞选，参与的玩家成为"警上玩家"，不参与的为"警下玩家"
2.  **警上发言**（每人120秒）：警上玩家按系统规定顺序发言，可选择"退水"（放弃竞选），退水后无被选举权
3.  **警下投票**（15秒）：警下玩家为警上玩家投票，退水玩家无投票权
4.  **警长确定**：
    - 得票最多者当选警长，拥有**1.5票投票权**和**最后发言**权
    - 平票则进入PK发言，再投票；再次平票则警徽流失，本局无警长
    - 狼人自爆可"吞警徽"：双爆（两天各一狼自爆）导致警徽流失

#### 4. 白天发言与投票阶段
1.  **公布死亡信息**：系统宣布昨夜死亡情况（平安夜/单死/双死），首夜死者有120秒遗言，其他夜晚死者无遗言
2.  **发言顺序**：
    - 无警长：夜晚一人出局从死者左侧/右侧开始；双死/平安夜随机顺序
    - 有警长：警长决定发言顺序，通常从警长左侧/右侧开始，警长最后发言
3.  **玩家发言**（每人120秒，警长150秒）：玩家依次发言，阐述自己观点，狼人可在发言阶段自爆
4.  **放逐投票**（15秒）：所有玩家投票，得票最多者出局，白天出局者有120秒遗言
    - 平票则进入PK发言，再投票；再次平票则无人出局
    - 警长投票视为1.5票，可决定关键局
5.  **出局结算**：
    - 猎人出局可发动技能枪杀一人（被毒杀除外）
    - 白痴被投票出局可翻牌免疫，之后可发言但不能投票
    - 暗牌局：玩家出局不翻开身份牌

#### 5. 循环与结束
- 重复"夜晚行动→白天发言投票"流程，直至某一阵营达成胜利条件
- 警长死亡时可选择移交警徽或撕毁警徽

---

### 四、特殊规则说明

#### 1. 自爆规则
- 狼人可在白天发言阶段任意时间自爆，打断当前发言，发表30秒遗言后直接天黑
- 自爆狼人若夜间已被毒杀，自爆后公布死亡信息进入白天放逐阶段
- 警上竞选阶段狼人自爆可影响警徽归属，双爆吞警徽

#### 2. 遗言规则
- 晚上死亡：仅首夜死者有120秒遗言，其他夜晚死者无遗言
- 白天死亡：所有被投票放逐或被猎人枪杀的玩家均有120秒遗言

#### 3. 时间设定
| 环节 | 时间 |
|------|------|
| 上警报名 | 15秒 |
| 玩家发言 | 120秒 |
| 警长发言 | 150秒 |
| 投票时间 | 15秒 |
| 遗言时间 | 120秒（白天/首夜），30秒（狼人自爆） |

---

### 五、关键注意事项
1.  女巫全程不可自救，每晚最多使用一瓶药
2.  猎人被毒杀或殉情时无法发动技能
3.  白痴被投票出局可免疫，但被刀或被毒杀会直接死亡
4.  警长拥有1.5票投票权和最后发言顺序，是好人阵营的关键领袖
5.  屠边规则：狼人只需消灭所有神职或所有平民即可获胜，无需屠城
