# 标准12人场（预女猎白）规则流程

12人标准场采用**预女猎白**板型，为**暗牌局**且**有警长**，采用**屠边**规则判定胜负。以下是完整规则流程：

---

### 一、基础配置

| 阵营 | 身份 | 数量 | 核心技能 |
|------|------|------|----------|
| **狼人阵营** | 普通狼人 | 4 | 每晚可袭击一名玩家，白天可自爆 |
| **好人阵营** | 预言家 | 1 | 每晚可查验一名玩家身份（好人/狼人） |
| | 女巫 | 1 | 拥有解药（救被刀玩家）和毒药（毒杀任意玩家），每晚最多用一瓶药，**首夜可自救，之后不可自救** |
| | 猎人 | 1 | 被狼人杀害或投票放逐时可翻牌枪杀一名玩家，**被毒杀不可开枪** |
| | 白痴 | 1 | 被投票放逐时可翻牌免疫出局，之后可发言但不能投票，狼人仍需击杀一次 |
| | 平民 | 4 | 无特殊技能，通过发言和投票帮助好人阵营 |

---

### 二、胜利条件

- **狼人阵营**：达成**屠边**——杀死所有神职（预言家、女巫、猎人、白痴）或所有平民
- **好人阵营**：放逐所有狼人出局

---

### 三、完整游戏流程

#### ① 游戏准备

12 人入座→全部准备→系统发牌→语音播报「游戏开始，天黑请闭眼」。

#### ② 夜晚阶段（固定顺序 + 时间）

1. 狼人阶段（40 秒）：睁眼→沟通选刀人目标→确认后闭眼；
2. 女巫阶段（30 秒）：睁眼→获知刀人目标→选解药 / 毒药 / 空过→闭眼（首夜可对自己使用解药自救）；
3. 预言家阶段（25 秒）：睁眼→选查验目标→获取结果→闭眼；
4. 白痴阶段（仅首夜 5 秒）：系统确认身份，无操作；
5. 猎人阶段（仅首夜 5 秒）：系统告知「可开枪状态」，无操作；
6. 黑夜结算：判定「刀人 + 解药 + 毒药」结果，记录单死 / 双死 / 平安夜。

> **注意**：夜间行动顺序为 狼人→女巫→预言家（网易标准规则）。首夜额外增加白痴和猎人的确认环节，非首夜跳过。女巫先于预言家行动，先获知刀口信息。

#### ③ 警长竞选阶段（仅第一天白天）

1. 上警报名（15 秒）：选「上警 / 不上警」，超时默认不上警；

2. 警上发言：**从最后一个上警的玩家开始逆序发言**，每人120秒发言时间，发言时可「退水」（放弃竞选，转为警下投票）；

3. 警下投票（15 秒）：**警下**玩家投剩余警上玩家（候选人不能投票），超时默认弃票；

4. 警长确定：

   - 得票最高者当选，获 1.5 票投票权 + 最后发言权；
   - 平票→PK 发言（60 秒）→重投；二次平票→警徽流失；
   - 双爆吞警徽：竞选阶段若两狼连续自爆，警徽永久流失（本局无警长）；仅一狼自爆则终止本轮竞选，当日无警长但不影响后续重选资格。

#### ④ 白天发言与投票阶段（循环至游戏结束）

1. 死亡播报（5 秒）：公布昨夜死亡情况，首夜死者有 120 秒遗言，其他夜晚死者无遗言；

2. 发言顺序：

   - 无警长：单死从死者下一位开始，双死 / 平安夜随机起始；
   - 有警长：警长定发言方向，自己最后发言（警长发言 150 秒 / 快速场 120 秒）；

3. 放逐投票（15 秒）：

   - 警长票计 1.5 票（面板显示 1 票，结算自动折算）；
   - 平票处理：平票玩家 60 秒 PK 发言→重投；二次平票→无人出局；

4. 出局结算：

   - 猎人出局：弹窗选枪杀目标（15 秒），被毒杀则无法开枪；
   - 白痴出局：自动翻牌免疫不出局，翻牌后可发言但不能投票；
   - 被猎人枪杀的玩家**无遗言**；
   - 所有出局者暗牌不翻身份。

#### ⑤ 循环与结束

重复「夜晚→白天」流程，警长死亡可「移交警徽 / 撕毁警徽」（超时则撕毁），直至某阵营达成胜利条件。

---

### 四、特殊规则说明

#### 1. 自爆规则
- 狼人可在白天发言/投票/竞选阶段自爆，打断当前发言，发表30秒遗言后直接天黑
- **已死亡的玩家不能自爆**
- 竞选阶段狼人自爆：单爆终止竞选（当日无警长），双爆吞警徽（本局永久无警长）
- 非竞选阶段自爆：自爆狼人发表遗言后跳过剩余白天直接进夜

#### 2. 遗言规则
- 晚上死亡：仅首夜死者有120秒遗言，其他夜晚死者无遗言
- 白天被投票放逐：有120秒遗言
- 被猎人枪杀：**无遗言**
- 狼人自爆：30秒遗言（仅非竞选阶段）

#### 3. 女巫规则
- 每晚最多使用一瓶药（不能同一晚既救人又毒人）
- 首夜可以对自己使用解药自救，之后不能自救
- 解药只能救被狼人袭击的玩家（刀口）
- 毒药可以毒杀任意存活玩家

#### 4. 猎人规则
- 被狼人杀害或被投票放逐时可以开枪带走一人
- 被女巫毒杀时**不能开枪**（即使有开枪能力，毒杀使技能失效）
- 首夜法官会确认开枪状态（仅首夜）
- 猎人夜间死亡时，等天亮公布死讯后才开枪

#### 5. 时间设定
| 环节 | 时间 |
|------|------|
| 上警报名 | 15秒 |
| 竞选发言 | 每人120秒 |
| 警长发言 | 150秒 |
| 投票时间 | 15秒 |
| 遗言时间 | 120秒（白天被放逐/首夜），30秒（狼人自爆），无（被枪杀/非首夜） |

---

### 五、关键注意事项
1.  女巫首夜可自救，之后不可自救，每晚最多使用一瓶药
2.  猎人被毒杀时无法发动技能，被狼杀或被放逐时可开枪
3.  白痴被投票出局可免疫，但被刀或被毒杀会直接死亡
4.  被猎人枪杀的玩家无遗言
5.  警长拥有1.5票投票权和最后发言顺序，是好人阵营的关键领袖
6.  屠边规则：狼人只需消灭所有神职或所有平民即可获胜，无需屠城
7.  夜间行动顺序：狼人→女巫→预言家（女巫先于预言家行动）
8.  竞选发言从最后一个上警的玩家开始逆序发言
9.  双爆吞警徽：竞选阶段两狼连续自爆，警徽永久流失
